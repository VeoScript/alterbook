# Alterbook

Old school, minimal social media app.

## Description

Bring back memories. A basic social media app using modern tech-stack.

## Getting Started

### Dependencies

#### Front-End (Mobile)
* React Native.
* Zustand
* Tanstack React-Query

#### Back-end (API)
* NestJS
* Prisma
* PlanetScale
* Render

### Installing

#### Both Mobile & API
```
yarn
```

### Executing program

#### Mobile (React Native)
```
yarn android
```

#### API (NestJS)
```
yarn start:dev
```
